Tabletop-style button (ink outline, hard offset shadow, presses down 2px); use variant="cambia" ONLY for the Cambia call or destructive acts, "gold" for ranked/win moments.

```jsx
<Button variant="primary" size="lg" onClick={fn}>Create lobby</Button>
<Button variant="cambia">Call Cambia</Button>
```

Variants: primary (ember), secondary, ghost, cambia (berry), gold (honey). Sizes sm/md/lg. `fullWidth` stretches.
